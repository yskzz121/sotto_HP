## 4. ブランドストーリー

### オーナーシェフ：潮平里志
- イタリアで4年半修業（フィレンツェ、ヴェネト州でオステリア立ち上げに参加）
- ティラミス発祥の地・北イタリア トレヴィーゾでレシピを学ぶ
- 沖縄の伝説的レストラン「ARDOR」（北谷）のシェフを経て独立
- 2021年：Sotto（ティラミス専門店）オープン
- 2022年秋：姉妹店 Totto（カウンターイタリアン）オープン

### Sottoのこだわり
- マスカルポーネチーズはイタリアから毎週空輸
- 材料：マスカルポーネ、卵黄、きび砂糖、少量のマルサラ酒
- 生クリーム・メレンゲ不使用（本場トレヴィーゾの製法）
- ココアパウダーは注文時にかけたて
- 素材の状態に合わせて毎回レシピを微調整

## 7. デザイン方針

### コンセプトキーワード
本物感 / 手触り感 / 素材感 / 職人感 / アーティスティック / 温かみ / クリエイティブ / ガストロノミー

### デザイン仕様

| 要素 | 方針 |
|------|------|
| 配色 | 温かみのあるアースカラー（クリーム、ブラウン、オフホワイト）＋アクセントにダークブラウン |
| フォント（見出し） | セリフ体（クラシック感・本物感） |
| フォント（本文） | サンセリフ体（可読性・モダン） |
| 写真の見せ方 | 大きく・余白多め。素材の質感が伝わるようノイズやオーバーレイは最小限 |
| アニメーション | スクロール時のフェードイン程度。控えめで上品 |
| レイアウト | レスポンシブ対応（PC / タブレット / スマホ） |
| 全体トーン | イタリアの田舎のトラットリアのような温かさ＋日本の職人の静けさ |
